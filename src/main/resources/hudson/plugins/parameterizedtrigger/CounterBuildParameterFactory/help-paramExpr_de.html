KEY=value Paare, eines pro Zeile (wie in einer Java Properties File). Backslashes werden zum Escapen benutzt, somit wird aus "\\" dann "\".

Build Parameter des aktuellen Builds und/oder Umgebungsvariablen können in folgender Form benutzt werden: ${PARAM} or $PARAM.

Der Ausdruck $COUNT oder ${COUNT} wird mit dem aktuellen Zähler ersetzt.
