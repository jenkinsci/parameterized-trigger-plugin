Groovy script that checks additional conditions on the build. Leave the script empty to disable any script execution.

The script takes several parameters:

  • build - Hudson Build
  • listener - Task listener, useful to log information

For example, to enable this trigger only at night, between 21:00 and 6:00, use the following script: 

hour = Calendar.instance.get(Calendar.HOUR_OF_DAY)
listener.logger.println "Executing script at ${hour}H"
(hour >= 21 || hour < 6)
