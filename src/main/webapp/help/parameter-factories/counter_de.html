Dieser Parametererzeuger erzeugt eine Folge von Zahlen als Parameter. Genauer wird von Von bis nach Bis mit Schrittweite Schrittweite gezählt, und für jede dieser Zahlen wird der Parameter-Abschnitt ausgewertet, wobei $COUNT durch den aktuellen Wert des Zählers ersetzt.

Für jeden dieser Parameter wird dann ein Build der ausgewählten Projekte gestartet. Wenn die Zähler leer ist, so wird kein Build gestartet.
